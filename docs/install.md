# Installation

FennFlow is available on PyPI as [fennflow](https://pypi.org/project/fennflow/) so installation is as simple as:

=== "pip"

    ```bash
    pip install fennflow
    ```

=== "uv"

    ```bash
    uv add fennflow
    ```

(Requires Python 3.10+)

This installs the `fennflow` package, core dependencies, and libraries required to use FennFlow.
